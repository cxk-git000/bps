# Investigation Notes

This directory contains short-form documentation written during the accumulator
bug investigation and repair process.

## Files

- `ACCUMULATOR_FIX.md`
  - concise note describing the accumulator-generation fix and the reasoning
    behind it

- `PBS_ACCUMULATOR_INVESTIGATION_REPORT.md`
  - longer narrative of the debugging process, observed symptoms, and repair
    rationale

## Why These Notes Matter

These files capture the path from observed anomalous outputs to the final repair
strategy. They are not just background notes; they explain why the later
guard-band and identity-check experiments were necessary.

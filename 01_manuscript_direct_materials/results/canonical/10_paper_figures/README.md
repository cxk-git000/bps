# 10 Paper Figures

This folder contains figures generated from the canonical result summaries.

## Format Policy

- `PDF` files are the primary figure files.
- `PNG` files are included for quick GitHub preview.
- Multi-series plots distinguish methods by both color and line/marker style rather than by color alone.

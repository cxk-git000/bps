# Paper Figures

This directory contains the manuscript figure set generated from the current
canonical results and method schematics.

## Files

- `fig01_system_overview`
  - shared input binning, shared codebooks, and scheme-specific plaintext embedding
- `fig02_input_encoding`
  - storage comparison for Standard PBS, SDR-PBS, and many-LUT
- `fig03_scheme_comparison`
  - structural comparison of Standard PBS, SDR-PBS, and many-LUT
- `fig04_guardband_layout`
  - concrete guarded and unguarded storage layouts
- `fig05_main_results`
  - main comparison across the supported function pairs
- `fig06_repair_effect`
  - pre-fix versus guarded comparison
- `fig07_timing_breakdown`
  - total latency and stage-level latency
- `fig08_guardband_ablation`
  - guard-band factor and offset study

Each figure is exported as both `PDF` and `PNG`.

# Paper Tables

This directory contains the compact table set derived from the current
canonical results.

## Files

- `tbl01_scheme_setup`
  - per-scheme parameter and layout summary
- `tbl02_main_results`
  - main comparison table across function pairs
- `tbl03_timing_summary`
  - stage-level timing summary
- `tbl04_codebook_summary`
  - codebook recovery summary

Each table is exported as both `CSV` and `LaTeX`.
